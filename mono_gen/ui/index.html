mono_gen readme

MonoX is a participative art engine based on StyleGAN-V. It learns from analog monotypes to simulate a visual latent evolution. Each walk through latent space generates one unrepeatable Mono. learn more >

How to Use

  1. Initiate Latent Walk → triggers generation
  2. Receive your Mono → stored temporarily
  3. Mint to Wallet → available for gas fee only
  4. Blend with Others → if eligible for current epoch
  5. No account creation required. All actions are public.
